# ⚖️ LexFlow

AI-assisted platform for creating structured legal document drafts through guided questionnaires.

## Vision

Help users prepare organized legal documents without replacing professional legal advice.

## Status

🚧 MVP Foundation

## Technology

- Python
- Flask
- HTML/CSS
